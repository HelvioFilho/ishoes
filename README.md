## Getting started

Clone the project and access the folder

```bash
$ git clone https://github.com/HelvioFilho/ishoes.git && cd ishoes
```

Follow the steps below
```bash
# Install the dependencies
$ npm install

# Start the project
# Android
$ expo run:android 

# IOS
$ expo run:ios
```
